# WindowsEnv
Dépôt pour mon environnement Windows 11. Permet après installation de Windows de personnaliser mon profil Windows et d'installer mes logiciels.

## Utilisation & Script
Lancement du script dans Powershell ISE en tant qu'administrateur. 
Si le poste vient d'être installé. Il est possible que l'exécution des scripts Powershell soit désactivée.
```
Set-ExecutionPolicy Unrestricted 
```

## Logiciel 
Installation des logiciels via Winget
### Logiciels minimals
- Firefox Developer Edition
- 7Zip
- Notepad++
- VLC
- Keepass
- Mozilla Thunderbird
- Discord
- Obsidian
- Adobe Acrobat Reader
### Logiciels optionnels
- Visual Studio Code
- WinSCP
- OBS
- Git
- Docker
